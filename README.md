# ⌬ Anxinity PowerFlux – Advanced Watt Control Engine ⌬

Redefining Android Power Flow.

Anxinity PowerFlux gives you precise control over charging wattage
(33W / 45W / 60W / 90W / 120W) with realtime watt monitoring, KernelSU
WebUI interface, adaptive node detection, and Magisk/KSU support.

## Installation
1. Download the ZIP release: `Anxinity-PowerFlux-v1.0.zip`
2. Flash via Magisk Manager (Modules -> Install from storage) or install via KernelSU.
3. Reboot device.
4. Open KernelSU WebUI or PowerFlux Web UI and choose profile.

## Notes
- This module uses best-effort sysfs node detection; behaviour depends on kernel/OEM.
- Use responsibly. Incorrect node writes may affect charging behavior.
